Criar um sistema para acessar os livros cadastrados em uma biblioteca virtual. Para isso, precisamos cadastrar os objets da nossa biblioteca:

- Nome para biblioteca
- Endereço para biblioteca


Um array chamado livros e vamos cadastrar tres objetos para cada livro:
- titulo 
- autor
- ano


Vamos criar uma funçao qu ira dos livros cadastrados. Esta funçao sera chamada  por meio de um botao que teremos na nossa tela frontend - HTML.

Vamos precisar:
UM arquivo HTML 
UM arquivo css
UM arquivo javascript
